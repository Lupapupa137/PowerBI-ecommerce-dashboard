# Power BI Dashboard — Анализ продаж и логистики e-commerce

**Проект:** Аналитический дашборд Power BI по данным интернет-площадки (Olist, Бразилия).  
**Цель:** показать навык работы с Power BI и визуализацией бизнес-метрик.

---

## Описание

Дашборд отражает ключевые метрики интернет-продаж:

- **Динамика выручки по месяцам** — рост весной, пик в июне  
- **Средний чек:** 137,75 R$  
- **ТОП-категории по выручке:** *health_beauty*, *watches_gifts*  
- **Среднее время доставки:** 12,5 дней  
- **География доставок:** плотность в юго-восточных регионах Бразилии  


| Использовано | Назначение |
|-------------|------------|
| **Power BI Desktop** | Построение визуализаций и расчёт метрик |
| **CSV-файлы (9 таблиц)** | Исходные данные e-commerce |
